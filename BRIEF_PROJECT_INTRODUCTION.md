# Clarity Mental Health App - Project Introduction

## Overview

**Clarity** is a comprehensive Flutter-based mental health application developed as my final year project. The app provides users with evidence-based tools to track, monitor, and improve their mental wellness through a secure, privacy-first platform.

## Key Features

- **Clinical Assessments**: PHQ-9 depression and GAD-7 anxiety screenings with progress tracking
- **Mood Tracking**: Daily mood logging with visual analytics and trend analysis
- **Digital Journaling**: Secure journaling with AI-powered sentiment analysis
- **Data Insights**: Comprehensive analytics dashboard with personalized recommendations
- **Privacy-First**: End-to-end encryption with Firebase backend integration

## Technology Stack

- **Frontend**: Flutter (Cross-platform mobile development)
- **Backend**: Firebase (Authentication, Firestore, Cloud Functions)
- **State Management**: Riverpod
- **AI Integration**: Google Generative AI for insights
- **Data Visualization**: FL Chart for mood trends

## Project Significance

This project addresses the growing need for accessible mental health tools by combining evidence-based psychological assessments with modern mobile technology. The app demonstrates proficiency in full-stack mobile development, healthcare data security, and user-centered design principles.

## Impact

Clarity empowers users to take control of their mental health journey through:
- Accessible, clinically-validated assessment tools
- Secure, private data management
- Actionable insights from personal mental health data
- Seamless cross-platform experience
