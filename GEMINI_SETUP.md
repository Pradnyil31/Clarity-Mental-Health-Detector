# Gemini AI Setup Instructions

Your Flutter app has been updated to use Google's Gemini AI model for chat functionality with emotion analysis. Here's how to set it up:

## Getting Your Gemini API Key

1. Go to [Google AI Studio](https://aistudio.google.com)
2. Sign in with your Google account
3. Click "Get API key" in the left sidebar
4. Create a new API key
5. Copy the API key

## Setting Up the API Key in Your App

1. Open your Flutter app
2. Navigate to the AI Chat screen
3. Tap the menu button (three dots) in the top right
4. Select "Gemini Settings"
5. Paste your API key and tap "Save API Key"

## Troubleshooting

- **"Failed to initialize Gemini AI"**: Make sure you've entered a valid API key in the settings
- **Empty responses**: Check your internet connection and API key validity
- **Slow responses**: This is normal for AI processing; responses typically take 2-5 seconds

## Privacy & Security

- Your API key is stored securely on your device
- Conversations are sent to Google's Gemini API for processing
- No conversation data is permanently stored by the app
- You can clear your API key anytime in the settings
